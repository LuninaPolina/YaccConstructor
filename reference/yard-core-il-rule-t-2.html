YaccConstructor

t<'patt, 'expr>

Namespace: Yard.Core
Parent Module: Rule

t<'patt,'expr> - Type of rule. 'patt - type of attributes (arguments). 'expr - type of expressions in action code. Rule have the following format: [+]name<< metaArgs >>[args] : body;
